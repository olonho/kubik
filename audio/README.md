# Папка для аудиофайлов

## Как добавить музыку "Only You" для катсцены победы

1. Скачайте файл "Only You" by The Platters (версия из Far Cry 5)
2. Переименуйте файл в `only-you.mp3`
3. Поместите файл в эту папку (`audio/`)

## Где скачать

Вы можете найти песню:
- На YouTube Music
- Spotify (экспортировать в MP3)
- Apple Music (экспортировать в MP3)
- Купить на iTunes/Amazon Music

## Формат

Игра поддерживает форматы:
- MP3 (рекомендуется)
- WAV
- OGG

## После добавления файла

Файл будет автоматически воспроизводиться в катсцене победы на 20 волне.

---

**Примечание**: Убедитесь, что файл имеет правильное имя: `only-you.mp3` (строчными буквами).
